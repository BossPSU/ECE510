If you are running complex tools or testing your code, please make
sure that you check for runaway processes. Running the command:

        ps -ef | grep $USER

should get you a list of everything you have going on the system. If
you have no other way to terminate a process, you can use:

        kill -9 PID

where PID is the process ID found from running ps as above. You can
also lower the priority (and thus the system impact) of a process
with this command:

        renice 15 PID

If you need to run a long running background job, send email to

        support@cat.pdx.edu

    letting us know when and where it is going to be run and its purpose.

For more information of running long background jobs see:

https://cat.pdx.edu/platforms/linux/administrivia/long-running-jobs-vs-runaways/
